Windows Shortcuts
To take snip of screen - screen Shot : Press and hold windows key + Shift + s
To add emoji from your keyboard : Windows logo key + period (.) Note: this period is not from NumPad
To open File Explorer quickly : Press and hold windows logo key + E
To get to your desktop quickly : Press and hold Windows logo key + D
To Open settings : Press and hold windows logo key + I
To Cut or copy or paste: Ctrl + C , Ctrl + X, Ctrl + V
To Switch between open windows : Alt + keep pressing Tab
